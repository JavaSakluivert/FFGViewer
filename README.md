# FFGViewer

株式会社大林組開発の「FINAL」で出力される `.ffg` ファイルの内容を表示します。

履歴ファイルに記述されている解析モデルの荷重−変位関係（`.ffg` 形式）を可視化する WPF デスクトップアプリケーションです。実験データ（`.xlsx` / `.csv`）との重ね合わせ比較にも対応しています。

## ダウンロード・起動方法

### ダウンロード

1. [GitHub Releases](https://github.com/JavaSakluivert/FFGViewer/releases) を開く
2. 最新バージョンの **Assets** から `FFGViewer-vX.X.X-win-x64.zip` をダウンロード
3. ZIP を任意のフォルダに展開する
4. `FFGViewer.exe` を実行する

> **動作環境**: Windows 10 / Windows 11 専用です。macOS・Linux では動作しません。
>
> **注意**: 本アプリはコード署名なし（自己ビルド）のため、.NET ランタイムのインストールは不要ですが、
> Windows SmartScreen の警告が表示されることがあります。

## 使い方

### ファイルの読み込み

以下の3つの方法でファイルを読み込むことができます。複数ファイルを読み込むと、グラフに重ね合わせ表示されます。

| 方法 | 操作 |
|------|------|
| **メニューから開く** | ツールバーの「ファイルを開く」ボタンをクリック → ファイル選択ダイアログで1つまたは複数のファイルを選択 |
| **ドラッグ＆ドロップ** | ファイルをウィンドウ上にドラッグ＆ドロップ（複数ファイル同時対応） |
| **コマンドライン引数** | `FFGViewer.exe path/to/data.ffg` のように起動（アイコンへの D&D も可） |

### 対応ファイル形式

| 形式 | 拡張子 | 用途 | 備考 |
|------|--------|------|------|
| **FFG** | `.ffg` | FINAL 解析結果 | Shift-JIS、スペース区切り、フッター `-999.0` |
| **Excel** | `.xlsx` | 実験データ等 | 最初のシートを読み込み。A列=変位、B列=荷重。ヘッダー行は自動判定（1行目が数値でなければスキップ） |
| **CSV** | `.csv` | 汎用データ | カンマ区切り or スペース区切り（自動判定）。1列目=変位、2列目=荷重 |

### グラフ操作

| 操作 | 説明 |
|------|------|
| **パン（平行移動）** | グラフ上でドラッグ |
| **ズーム** | マウスホイールで拡大・縮小 |
| **ズームリセット** | 右パネルの「ズームリセット」ボタン |
| **グラフクリア** | ツールバーの「グラフクリア」で全シリーズを削除 |

### データ確認・出力

| 機能 | 説明 |
|------|------|
| **ピークデータ表示** | 右パネルに各シリーズの最大/最小荷重・変位をテーブルで色分け表示 |
| **クリップボードコピー** | 「ピークデータをコピー」→ タブ区切りテキストとしてコピー（Excel に貼り付け可能） |
| **Excel エクスポート** | PeakData シート + データシート + XY 散布図グラフシートを含む `.xlsx` を出力 |
| **CSV エクスポート** | 全シリーズを列形式（`Disp_[名前], Load_[名前]`）で 1 ファイル出力 |
| **軸タイトル編集** | 右パネル下部でX軸・Y軸のタイトルをリアルタイム変更 |
| **間引き表示** | 15,000点超のデータをサイクル認識型 LTTB で間引き描画（ピーク計算・エクスポートは元データ使用） |

## 対応ファイル形式 詳細

### .ffg ファイル（FINAL 解析結果）

```
（空行）
タイトル行（最大5文字、超過分は切り詰め）
変位1 荷重1
変位2 荷重2
...
-999.0 -999.0    ← フッター（データ終端）
```

- **エンコーディング**: Shift-JIS
- **区切り文字**: 半角スペース

### .xlsx ファイル（実験データ）

```
A列(変位)  B列(荷重)
 0.00       0.00       ← ヘッダーなし: 1行目からデータ
 1.00      10.50
 2.00      20.00
...
```

- **対象シート**: 最初のシートのみ
- **ヘッダー自動判定**: 1行目のA列・B列が両方とも数値ならデータとして扱い、どちらかが文字列ならヘッダーとしてスキップ
- **空行・非数値行**: 自動スキップ
- **シリーズ名**: ファイル名（拡張子なし）

### .csv ファイル

```
0.00,0.00
1.00,10.50
2.00,20.00
...
```

- **区切り文字**: カンマ優先、カンマがなければスペース（自動判定）
- **空行・非数値行**: 自動スキップ
- **シリーズ名**: ファイル名（拡張子なし）

## ビルド・テスト（開発者向け）

### 前提条件

- [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0)
- Windows 10/11

### ビルド

```bash
dotnet build FFGViewer/FFGViewer.csproj
```

### 実行

```bash
dotnet run --project FFGViewer/FFGViewer.csproj
```

コマンドライン引数としてファイルパスを渡すと起動時に読み込まれます。

```bash
dotnet run --project FFGViewer/FFGViewer.csproj -- path/to/data.ffg path/to/experiment.xlsx
```

### テスト

```bash
dotnet test FFGViewer.Tests/FFGViewer.Tests.csproj
```

**テスト結果**: 73件 / 73件 合格
